Dagre D3 Demo: Style Attributes

An example showing how styles that are set in the input graph can be applied to nodes, node labels, edges, and edge labels in the rendered graph.

Node attributes prefixed with "style" (e.g. "styleStrokeWidth") are uncamelized (e.g. "stroke-width") and applied to the node shape. The same applied for "style" prefixed attributes on edges and their associated paths. Nodes or edge attributes with the prefix "labelStyle" are similarly applied to the labels for the nodes or edges.
